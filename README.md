# Southwest Mining & Processing Solutions Website

Custom static website package for southwestminingandprocessingsolutions.com.

## Files
- index.html
- styles.css
- assets/logo.jpeg

## Deploy on Vercel
1. Create a GitHub repository named `southwest-mining-processing-site`
2. Upload these files to the repository
3. Go to Vercel.com
4. Continue with GitHub
5. Import the repository
6. Click Deploy
7. Add your domain: southwestminingandprocessingsolutions.com
